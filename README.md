# StageMaroc

Application web de gestion des stages développée en PHP/MySQL.

## Description

StageMaroc est une plateforme permettant aux stagiaires de rechercher des offres de stages et aux entreprises de publier et gérer leurs offres.

## Fonctionnalités principales

### Espace Stagiaire
- Création de compte
- Connexion sécurisée
- Gestion du profil
- Consultation des offres de stages
- Envoi de CV

### Espace Entreprise
- Création de compte entreprise
- Publication d'offres de stages
- Gestion des offres publiées
- Consultation des candidatures

## Technologies utilisées

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
